# Unit Converter ⚡

Angular app for converting units.

## 🚀 Run Locally

### Without Docker

```sh
npm install
npm run start
```

Open [http://localhost:4200](http://localhost:4200)

### With Docker 🐳

```sh
docker build -t unit-converter .
docker run -p 8080:8080 unit-converter
```

Open [http://localhost:8080](http://localhost:8080)

Happy converting! 😃
